# Jude Pascal — Today is the Day

Plain HTML / CSS / JavaScript site. No Node, no build step, no WebGL.

## Run it

**Option A — open in browser**

Double-click `index.html`, or drag it into Chrome/Safari.

**Option B — GitHub Pages**

Push this folder to a GitHub repo and enable Pages. That is how `judepascal.com` will be hosted.

## What is in this folder

```
index.html          — page
css/style.css       — layout
js/player.js        — cassette player (audio + buttons)
assets/             — images, audio, favicon (all local)
```

Nothing here depends on Google Sites, TurboWarp, or npm.

## Deploy

1. Push **only** this folder to GitHub (`main` branch, root).
2. Enable GitHub Pages.
3. Point `judepascal.com` DNS to GitHub Pages.

Do **not** upload `node_modules` — there is none. The site is static files only.
